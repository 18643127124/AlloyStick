How Easy Is It

既然是动画，我们就不能单靠编码的力量去完成。我们第一步是去动画编辑器里面，制作我们的骨骼动画。只需要在编辑器里绑定骨骼关系，设置简单的动画关键帧，依靠我们强大的自动补间和骨骼关系，就可以制作出生动canvas动画，把它运行在你的PC，你的平板和你的手机里。可惜的是，我们骨骼动画编辑还在努力完善中。


var canvas = document.getElementById('canvas'),
	textureImg = document.getElementById('knightImg');

var stage = new alloyge.Stage(canvas.getContext('2d'));
var	armature = new alloysk.Armature('knight',textureImg);

// animationName ,totalFrames,transitionFrame,isLoop
armature.playTo('run',40,15,true);
armature.setPos(150,300);	

stage.addObj(armature);

alloyge.monitorFPS(stage);  // 启动FPS监听器 (辅助功能 非必须)

// 开始场景里的动画，并且可以传入callback循环调用
						stage.start();
